在一个坐标系中画了两条实线和一条虚线，
问这三条线围成的“中间空白区域”如何用几个不等式表示
    这类题目属于平面区域的不等式表示

   用不等式表示由两条实线和一条虚线围成的空白区域的方法：

    1. 写出三条直线的方程  
    - 例如：实线1为 y = f1(x)，实线2为 y = f2(x)，虚线为 y = f3(x)（也可能是 x = 常数）。

    2. 判断每条直线对应的不等式方向  
    - 在图中观察空白区域位于该直线的哪一侧（上方/下方，左侧/右侧）。  
    - 可用测试点法：选一个明显在空白区域内的点（如 (0,0) 或其他易算点），代入直线方程。  
        • 若点满足 y > f(x)，则区域在直线上方，用 y > f(x) 或 y ≥ f(x)；  
        • 若点满足 y < f(x)，则区域在直线下方，用 y < f(x) 或 y ≤ f(x)。  
    - 对竖直线 x = c：  
        • 区域在右边 → x > c 或 x ≥ c；  
        • 区域在左边 → x < c 或 x ≤ c。

    3. 根据线型确定是否包含等号  
    - 实线：包含边界 → 用 ≤ 或 ≥  
    - 虚线：不包含边界 → 用 < 或 >

    4. 联立所有不等式，即为所求区域的数学表示。

    示例：  
    - 实线1：y = x + 2，空白区域在其下方 → y ≤ x + 2  
    - 实线2：y = -x + 4，空白区域在其下方 → y ≤ -x + 4  
    - 虚线：y = 1，空白区域在其上方 → y > 1  

    最终不等式组为：  
    y ≤ x + 2  
    y ≤ -x + 4  
    y > 1
