* P x t = E
| 
| * 电功率 x 时间秒数 = 能量
| 
| P 电功率 watts 
| t 时间 秒
| E 能量 焦耳J
| 

* I x V = P
| 
| * 电流 x 电压 = 电功率
| 
| I 电流current 单位A
| 

The current in the motor is 2.0 A and 
the battery has an electromotive force (e.m.f.) of 12 V. 
In a time of 8.0s, the motor does 140 J of work.

Calculate the energy supplied to the motor in 8.0s,
| 
| 题目给出的信息如下：
| 
| 电流（I） = 2.0 A
| 电动势（e.m.f. 或电压 E） = 12 V
| 时间（t） = 8.0 s
| 电机对外做的有用功（work done by the motor） = 140 J（这个是输出能量）
| 
| 计算在 8.0 秒内供给电机的能量（即输入的电能）
| 
| 使用电功率公式计算输入能量
| 供给电机的电能（即输入能量）可以用下面的公式计算：
| 
| Energy supplied= Power × Time
| 
| 而电功率（输入功率）为：
| 
| P=I×V
| 
| 代入数值：P=2.0A×12V=24W
| 
| 然后计算 8.0 秒内的能量：
| Energy supplied= 24W × 8.0s = 192J
|

电机的效率
| 
| * 输出能量 / 输入能量
| 


重力势能 Gravitational Potential Energy
| 
| 重力势能是物体由于被举高而具有的能量。单位 J
|
| 当你把一个物体抬高时，你对它做了功，
| 这个能量被“储存”起来了，称为重力势能。
| 当物体下落时，这种能量可以转化为动能。
| 
| * 重力势能 = m x g x h
| 
| m = 质量
| g = 重力加速度
| h = 上升的高度 
| 

功率 power
| 
| 做功的快慢
| 或 能量转移的速率。
| 是一个物理概念
| 
| 单位 W Watts
| 
| 1 W = 1 J / s 
| 
| 1瓦特 = 每秒1焦耳
| 

​功(the work done) = 力 × 位移 × cos(力与位移之间夹角)
| 
| 如果驱动力方向与运动方向相同(如开车)，
| 最后一项为1
| 
| 那么功(the work done)为 
| 力 x 距离
| 

P = W / t
| 
| 功率 = 做的功 / 所用的时间
| 
| P = power 功率 单位瓦特
| W = work 功 单位焦耳
| 

kinetic energy 动能
| 
| 动能，单位焦耳 J
| 
| 0.5 x m x v^2
| 
| 0.5 x 质量(千克) x 速度(m/s)的平方
